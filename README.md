# Airline Review Analytics in Python

Overview

This project analyzes airline customer reviews using Python.
It explores customer ratings, recommendation outcomes, and simple text-based review patterns.

Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn

What This Project Shows

* Data cleaning and preparation
* Exploratory data analysis
* Customer recommendation analysis
* Basic text analytics
* Predictive modelling workflow

How to Run

Open and run:

airline_review_analytics.py
